## Codebase Patterns
- Run tests with `.venv/bin/pytest` (not `python -m pytest`) — system Python on macOS lacks pytest
- Always install both requirements.txt AND requirements-dev.txt: `validators` is used in src but only in dev requirements
- src/ modules import each other directly (no package prefix) — add `sys.path.insert(0, "src/")` in test files
- conftest.py fixtures: `tmp_config` returns str path; `tmp_podcast_dir` returns pathlib.Path
- sample_feed.xml has 3 episodes; episode 2 title contains `/`, `:`, `?` for sanitization testing
